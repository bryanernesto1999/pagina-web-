¿Qué son las metodologías de nombrado?

Las metodologías de nombrado en desarrollo web ayudan a mantener el código organizado, predecible y fácil de mantener. Son especialmente útiles al trabajar con CSS y HTML, donde una estructura clara puede evitar conflictos de estilo.

BEM (Block Element Modifier)

BEM es una metodología que divide los nombres de clase en bloques, elementos y modificadores. Esto permite mantener un sistema de clases escalable y reutilizable.

  • Bloque: Representa un componente independiente.
  • Elemento: Parte del bloque que cumple una función específica.
  • Modificador: Representa una variación del bloque o elemento.
<div class="menu">
  <ul class="menu__lista">
    <li class="menu__item menu__item--activo">Inicio</li>
  </ul>
</div>

camelCase

camelCase es una convención de nombrado usada comúnmente en JavaScript, aunque también se puede aplicar a clases CSS. Consiste en escribir las palabras sin espacios y con mayúsculas en cada inicio de palabra después de la primera.

  • Ejemplo: nombreDeClase
  • Fácil de leer y escribir.
  • Menos común en grandes estructuras CSS.
<div class="tarjetaPrincipal">
  <p class="textoDestacado">Ejemplo camelCase</p>
</div>

Aplicación de conceptos HTML y CSS

A través de los ejemplos anteriores, se aplican los siguientes conceptos:

  • Estructura web: uso de <div>, <h1>, <ul>, <p>.
  • Propiedades de texto: font-size, color, text-align.
  • Listas: <ul> y <li> con estilos personalizados.
  • Espaciado con margin y padding.
  • Uso de border para resaltar secciones.
